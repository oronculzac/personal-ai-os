This directory contains example Excel files generated by the skill.

## Available Examples

### sales_report.xlsx
A sample sales report with:
- Product sales data
- Quantity and pricing
- Calculated totals (using formulas)
- Currency formatting
- Professional styling

Generated with:
```powershell
python .agent/skills/excel_generator/scripts/excel_builder.py --template sales_report --output sales_report.xlsx
```

## Creating Your Own Examples

Use the excel_builder.py script to create custom spreadsheets:

```python
from excel_builder import ExcelGenerator

# Create generator
gen = ExcelGenerator("my_report.xlsx")

# Add headers
gen.add_headers(["Name", "Value", "Total"])

# Add data
gen.add_data([
    ["Item 1", 100, "=B2*1.1"],
    ["Item 2", 200, "=B3*1.1"]
])

# Save
gen.save()
```
