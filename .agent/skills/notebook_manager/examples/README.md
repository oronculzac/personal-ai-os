# Notebook Manager - Examples

This directory contains example Jupyter notebooks generated by the Notebook Manager skill.

## Available Examples

### 1. Data Exploration Notebook
`data_exploration_example.ipynb` - Template for exploring datasets with:
- Data loading and overview
- Missing value analysis
- Statistical summaries
- Visualizations
- Insights section

### 2. Spark Development Notebook
`spark_job_example.ipynb` - Template for Spark job development with:
- Spark session setup
- Data loading from various sources
- Transformation examples
- Data quality checks
- Output writing

### 3. SQL Query Notebook
`sql_queries_example.ipynb` - Template for BigQuery/SQL development

### 4. ETL Pipeline Notebook
`etl_pipeline_example.ipynb` - Template for ETL development

## Usage

Create a new notebook:

```bash
# Data exploration notebook
python ../scripts/notebook_manager.py \\
    --template exploration \\
    --title "My Analysis" \\
    --output ./notebooks

# Spark development notebook
python ../scripts/notebook_manager.py \\
    --template spark \\
    --title "Spark Job" \\
    --output ./notebooks
```

## Best Practices

1. **Naming:** Use date prefix (YYYYMMDD_) for chronological order
2. **Organization:** Store in module-specific folders
3. **Version Control:** Clear outputs before committing
4. **Documentation:** Add markdown cells to explain logic
5. **Code Quality:** Keep cells focused and well-documented
